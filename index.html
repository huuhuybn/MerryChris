Employees System

Python Management Application Specification

Project Overview

The Employees System is a Python-based console application designed to manage employee records within an organization. The objective is to create a robust system that allows for the manipulation of employee data through a user-friendly command-line interface (CLI). This project focuses on using core Python concepts such as lists, dictionaries, functions, and control flow.

Core Features

1. Add New Employee

The system must allow the administrator to input details for a new employee and store them in the database.

  • Input Requirements: Unique Employee ID, Full Name, Age, and Salary.
  • Validation: Ensure the ID is unique and Age/Salary are positive integers.

2. List Existing Employees

Display a comprehensive list of all employees currently stored in the system.

  • Format: The output should be formatted as a table or a clear list for readability.
  • Example: ID: 101 | Name: John Doe | Age: 30 | Salary: $50,000

3. Delete Employees by Age Range

Ideally suited for bulk administrative tasks, this feature facilitates removing employees who fall within a specific age demographic.

  • Input: Minimum Age and Maximum Age.
  • Logic: Iterate through the list and remove records where:
    min_age <= employee_age <= max_age

4. Update Salary by Name

A specific utility to adjust the compensation of an employee identified by their name.

  • Input: Employee Name and New Salary amount.
  • Behavior: Search for the employee by name. If found, update the salary field. If multiple employees have the same name, consider asking for ID or updating all (specify in implementation).

Technical Requirements

Data Structures

Use Lists of Dictionaries or Classes to represent employee data.

Interface

Implement a while loop menu system to keep the program running until the user chooses to exit.

Sample Menu Structure:

1. Add New Employee
2. Show All Employees
3. Delete Employees (Age Range)
4. Update Salary
5. Exit
